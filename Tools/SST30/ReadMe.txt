Software Signature Tool (SST).
------------------------------
This tool allows to compute the software signature  and to generate files to be downloaded in the terminals with TELIUM ™ technology.

Environment:
------------
SST30 Software runs under Windows ® platform (NT4, 2000, XP). Computer must have a CR-ROM drive , a floppy disk drive and a serial port or an USB port.

Installation Process:
---------------------
See SST30 user's guide.


SST30 user's guide:
------------------
It describes how install,use and uninstall the tool. It may be consulted from the task bar click on "Start" button and choose "Programs-> TELIUM Tools -> SST30 user's guide" or from the CD-ROM (file SST30 User's guide.pdf).
