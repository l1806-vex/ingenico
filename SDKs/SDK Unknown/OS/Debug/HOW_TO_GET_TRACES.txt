HOW TO GET THE TRACES
----------------------------

To get the traces you have to load the file "System.Cfg". 
Please modify this file to select the communication port
Modify the value of the variable 'TRACE-DEV' as folow 
# TRACE_DEV : 0=no trace  1=USART0/COM1 2=USART1/COM2 3=USART2/COM0 4=USART3 5=USB_DEV
