---------------
 DLL SECURITY NumEntryGen SAMPLE
---------------

This sample shows how to use new numerical entries functionalities provided by the SECURITY DLL.
This sample is an application that uses messages provided by a messages file loaded in the terminal (MESS0201 file)

To use this sample, please perform the following steps.

1- Compile and load the sample in the terminal.

2 - Load the messages file (MESS0201.*)  in the terminal.
