FTP SAMPLE
---------------

This sample must be only executed on Internet terminals (Ethernet terminals : EFT30+ or Smart+,
or GPRS terminals : ETF930G, EFT930B ).

It shows how to perform a FTP transfer.


From the idle screen, press the 'F' key and select the item "Sample FTP" 
to enter the application.

It displays the folowing menu

+-----------------------+
| 0 - GPRS		|
| 1 - FTP		|
|			|
|			|
+-----------------------+


0 - GPRS

This function connects the GPRS network. It allows the input of the needed parameters :
APN, PPP login and password. By default, it displays the parameters entered from the
manager menus (TELIUM Manager/Initialization/Parameters/Network Access/GPRS).

1 - FTP

This function performs a FTP transfer. It connects a FTP server and downloads the content
of a remote folder. It allows the input of the parameters needed for the transfer 
(IP address and port, FTP login and password and the remote folder name).
