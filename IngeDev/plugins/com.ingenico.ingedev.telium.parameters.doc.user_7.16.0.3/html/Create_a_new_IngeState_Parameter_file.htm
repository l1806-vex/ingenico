Create a new IngEstate Parameter Management file (PML)

An IngEstate Parameter Management File (PML) is a XML file which is based on a specific XML schema.


To create a new PML file, select the project directory where you want to create the file and select the'New' item in the 'File' menu and then Other...":

 



In the new window, select Telium/IngeState/PML File :

 



Then enter the name of the file to create on Source File.

 


If you press the "browse" button then you can browse to another directory :

 


 

You should give a pml extention to the filename.


You can choose a template example or a simple empty pml file :


With the Finish button you have now access to your file.
