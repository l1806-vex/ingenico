<tfoot> element

This tag defines the footer portion of a TML table.

It should be used in conjunction with the <thead> and <tbody> elements.

Parent elements:

table

Attributes

NameTypeDescription
idxs:ID

Document-wide unique id. In practice only used for <screen> elements. Maximum length of a screen ID is 32 characters.

classxs:NMTOKENS

Space separated list of classes. Used for CSS formatting.

alignxs:token

Horizontal alignment attributes for cell contents. Note that you can use CSS properties instead.
The value should be one of the following:

  • left
  • center
  • right

valignxs:token

Vertical alignment attributes for cell contents. Note that currently you can not set this property via CSS.
The value should be one of the following:

  • top
  • middle
  • bottom

Child elements:

[tr+]
