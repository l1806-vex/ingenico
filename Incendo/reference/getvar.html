<getvar> element

This element returns the value of the variable specified by the name attribute. This value can be formatted using the format attribute.

Note that you must have the appropriate read permissions to access the variable.

See the variables tutorial for a detailed explanation and examples of TML variable use.

Parent elements:

a baddata dd display div dt form h1 h2 h3 layout li p pinentry pre print prompt span strtemplate submit td th txnsubmit

Attributes

NameTypeDescription
namexs:NMTOKENS

Required attribute.

Specifies the name of the referenced variable.

formatValrefFormatter

Defines a formatter pattern for the variable value. The pattern must be valid for the type of the variable you are trying to access.

The value of the variable will be transformed according to this pattern before being displayed/printed.

Note that the contents of the format attribute can be either be a valid formatter or a reference to a string - type variable that contains the pattern

The same as Formatter, but the format pattern can be specified either as a value string or as a variable reference to a string variable that contains the pattern. The referenced variable name must be prefixed by tmlvar: i.e. tmlvar:variable.name

Child elements:

No child elements.
